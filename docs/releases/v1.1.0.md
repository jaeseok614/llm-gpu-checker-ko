AI Hardware Fit가 단일 LLM 계산기에서 AI 워크로드 계획 도구로 확장되었습니다.

**[웹에서 바로 사용하기](https://jaeseok614.github.io/llm-gpu-checker-ko/)**

## 주요 변경사항

- 생성형 LLM 외 임베딩·리랭커·OCR·문서 VLM·범용 VLM 지원
- 총 286개 AI 모델과 90개 GPU 프리셋
- 모델 2~3개 비교
- 출처가 연결된 대표 공개 평가와 한국어 라이선스 이용 조건
- Hugging Face 공개 모델 직접 계산
- 이기종 GPU 병렬과 여러 GPU 모델 배치
- Ollama·llama.cpp 로컬 벤치마크 수집 CLI

## 함께 개선된 부분

- `빠른 추천`과 `전체 모델 탐색`을 최상위 모드로 분리
- GPU 설정 안에 Hugging Face 직접 계산과 여러 GPU 배치를 고급 도구로 배치
- 비교 화면을 항목별 고정 열로 정리하고 서로 다른 벤치마크의 직접 비교를 차단
- VRAM·처리량·동시 요청 계산 근거와 추정 신뢰도 표시
- 계산 추정·외부 공개 참고값·사용자 측정·자체 측정을 구분
- 핵심 계산식과 UI 상태에 대한 jsdom 회귀 테스트 추가

## 지원 규모

| 항목 | 수량 |

| GPU 프리셋 | 90 |

## 벤치마크 CLI

CLI는 실행 중인 로컬 Ollama 또는 llama.cpp 서버에만 연결해 prefill/decode 속도, TTFT, peak VRAM을 측정하고 제보용 JSON을 만듭니다.

```bash
node scripts/benchmark-cli.mjs --runtime ollama --model qwen3:8b --context 8192
```

측정 결과 JSON은 [`Benchmark report`](https://github.com/jaeseok614/llm-gpu-checker-ko/issues/new?template=benchmark-report.yml)에 직접 제출할 수 있습니다. CLI는 결과를 외부로 자동 전송하지 않습니다.
